Instructions to use the program

This program can perform some of the next operations

The program has a menu of buttons the button Game has the next buttons:

Button play

The button play closes the current game and it opens a new game

Button restart

Button restart closest the current window and it opens a new game with the same game

Button numbered

The button numbered shows a window with the last 6 games played

and it asks the user to type one of those games

Sound

Sound has two buttons to turn on or turn off the sound

Undo Button

It restores the last two removed tiles

Operation Button

It shows this instruction of how to use the application

Game Rules Button

It shows the rules to use the game
